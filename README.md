This project presents an end-to-end analysis of Reliance Industries Limited (NSE) stock using Power BI.
The dashboard visually tracks price movements, moving averages, trade volume, OHLC changes, daily volatility, and 6-month high/low comparisons, enabling better decision-making and trend analysis.

1. Project Overview

The goal of this project is to analyze the stock performance of Reliance Industries using historical market data and create an interactive Power BI dashboard.

This dashboard helps in:

Understanding short, medium, and long-term price trends

Analyzing volume & trade patterns

Measuring price volatility

Tracking the 6-month high/low and recent performance

Studying month-wise OHLC movement

Identifying price increase/decrease stages using waterfall analysis

This project reflects real-world financial analytics, beneficial for equity research, portfolio analysts, and data analysts.
